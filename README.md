# makeHeaders
===========

A very basic C/C++ header file generator from code and viceversa. The current version support bidirectional conversion of C and C++ files

----------
## Conversion features
---------

- Generate .h from .c
- Generate .h from .cpp
- Generate .c from .h
- Generate .cpp from .h

##### Drag & Drop selected file to convert it!
